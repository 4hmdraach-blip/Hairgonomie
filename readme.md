# Hair'gonomie – Menu adaptatif vivant

## Concept ergonomique

Le menu s’adapte automatiquement selon l’usage de l’utilisateur :

- Les options les plus utilisées avancent et attirent l’attention.
- Les options secondaires se déplacent ou s’estompent légèrement.
- La navigation devient fluide, surprenante mais intuitive.

## Problème utilisateur résolu

- Les menus classiques affichent trop d’options simultanément, ce qui surcharge l’utilisateur.
- Ici, la priorité visuelle réduit la confusion et facilite la navigation.

## Comment ça fonctionne

1. Cliquer sur une option déplace cette option au premier plan.
2. Les autres options se réorganisent aléatoirement pour créer un effet “vivant”.
3. L’interface est responsive et fonctionne sur mobile et desktop.

## Priorités et compromis

- Priorité : fluidité et mise en avant des éléments importants.
- Compromis : l’utilisateur doit s’adapter à la dynamique du menu, mais les animations sont douces pour ne pas gêner.

## Inspirations

- Micro-interactions Material Design
- Interfaces organiques et adaptatives
- Métaphore “chevelure vivante” pour visualiser la hiérarchie et le mouvement naturel
